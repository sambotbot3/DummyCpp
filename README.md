# DummyCpp (Dpp)

DummyCpp, or Dpp, is a small C++ to C transpiler project.

The goal is not to support all of C++ immediately. The goal is to make a useful, expanding subset of C++ lower to readable C, compile with a normal C compiler, and grow feature coverage over time.

See `docs/` for the initial plan, roadmap, decisions, resources, and condensed questions.
